El problema

🔍

Detección tardía

Las enfermedades suelen identificarse cuando ya han afectado grandes extensiones de cultivo, haciendo difícil su control.

👁️

Inspección manual

Depende de la experiencia de cada persona. Factores como la iluminación y el fondo complican el análisis visual.

💸

Pérdidas económicas

Para pequeños productores, sin herramientas automáticas accesibles, el daño en cultivos puede ser devastador.

Nuestra propuesta

Comparamos dos métodos sobre las mismas imágenes para entender cuál es mejor, cuándo y a qué costo computacional.

Método 1

Pipeline clásico

Sin supervisión — no necesita datos etiquetados

  1. Preprocesamiento (filtros Gaussiano y pasa alto)
  2. Análisis de color en espacio HSV
  3. Segmentación K-means (k=3 clusters)
  4. Morfología: apertura → cierre → erosión → dilatación
  5. Análisis de componentes conectados
✦ Aporte original del proyecto
VS
Método 2

Red neuronal (CNN)

Con transfer learning sobre dataset etiquetado

  1. Base: MobileNetV2 preentrenado en ImageNet
  2. Fine-tuning con imágenes de hojas
  3. Clasificación: sana vs enferma
  4. Grad-CAM para visualizar zonas activadas
  5. Evaluación: F1, accuracy, IoU
