📌 Nota sobre esta galería

Los recuadros grises de arriba son marcadores de posición. Para mostrar los resultados reales, ejecuta python demo_local.py en la carpeta backend/, copia las imágenes generadas en outputs/ a la carpeta frontend/img/ y reemplaza cada gallery-placeholder por una etiqueta <img>.

En la versión final de la web, este paso será automático: el backend enviará las imágenes procesadas directamente al frontend vía API REST.
